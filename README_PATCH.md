# CastorWorks 0.34.0 Commercial Contracts Milestone

This package represents the complete `release/0.34-commercial-contracts` milestone built for the public `v0.33.5` baseline.

It contains direct replacement and new files only. There is no source-patching script.

## Included capability

- Commercial account contract portfolio
- Multi-location service assignments
- Recurring service scheduling
- Recurring job generation
- Recurring contract billing
- Contract SLAs
- Renewal handling
- Amendments and version snapshots
- Purchase orders
- Compliance records
- Public agreement review and signature compatibility
- Worker run history and health evidence

## Important deployment note

The replacement `AgreementController.php` and agreement views use the existing routes already present in `public/index.php`. No router replacement is required.
